FreshLens

Point your camera at a fridge, basket, or produce shelf—and every item lights up at once with a live, color-coded freshness score. Then AI tells you what to do about it: rescue what’s turning, or mark it down before it’s lost.

Python FastAPI PyTorch YOLO / Ultralytics Claude API JavaScript

The leap

Expired Solutions already scans one item at a time and scores its freshness 0–100 across 135 produce types. FreshLens is the 10x: scan the whole scene at once. One tap over a fridge or a produce display, and every item is found, scored, and ranked—live.

That single capability serves both sides of the business. Pointed at a shopper’s fridge, it’s a magic “what do I eat first, and what can I cook tonight?” moment. Pointed at a store’s produce section, it’s an instant freshness audit that tells the manager exactly what to mark down—the heart of the markdown-automation pitch.

How it works

Three capabilities, built in sequence over the 4-week sprint.

1

Multi-Item Detection

A pretrained object detector (YOLO / Ultralytics) finds every produce item in the frame and returns a bounding box for each—no model training from scratch required.

2

Live Freshness Overlay

Each detected item is cropped and scored 0–100 by the freshness model, then drawn back onto the camera view with a green / amber / red overlay.

3

Act On It

Claude turns the scored scene into action: an “eat me first” ranking plus a recipe to rescue what’s turning (shopper), or a markdown worklist and shrink-savings estimate (store).

Architecture

A FastAPI service that chains pretrained vision with the freshness model and an LLM action layer. Beginners assemble proven parts rather than inventing them—exactly the skill the program teaches.

4-week build plan

Maps to program Weeks 5–8. Each Friday ends with a live demo. Hero path is the shopper experience; store manager-mode is the Week-8 stretch.

Week 5 · Detect + score one frame
  • Image upload endpoint (HEIC/JPEG/PNG)
  • Run a pretrained detector on a fridge/shelf photo
  • Crop each item; score freshness 0–100
  • Return boxes + scores as JSON
Demo: upload a fridge photo → every item boxed with a freshness score.
Week 6 · The live overlay
  • Mobile camera capture in the browser
  • Draw color-coded overlays (green/amber/red)
  • “Eat me first” ranking of the scene
  • Smooth the experience to near real-time
Demo: aim at a basket → scores appear live, worst-first list updates.
Week 7 · Claude action layer
  • Send detected items + scores to Claude
  • Generate a rescue recipe from what’s on hand
  • Explain “why this first” in plain language
  • Handle messy / partial detections gracefully
Demo: scan a fridge → “Use the spinach & tomatoes tonight—here’s a recipe.”
Week 8 · Store mode + harden
  • Store stretch: shelf photo → markdown worklist
  • Projected shrink $ saved from the scored scene
  • Error handling + 10+ unit tests
  • Docs: API contract + setup
Demo: scan a produce shelf → “Mark down these 6 items, save ~$48 today.”
